## 3. 구현 완료된 기능

### 진단 핵심 흐름
1. ✅ 면책조항 동의 게이트 (체크박스 + 시작 버튼)
2. ✅ 사업자 상호 / 본인 성함 / 이메일(필수, 형식 검증) 입력
3. ✅ 주력 매물 10종 복수 선택 + 기타 자유 입력
4. ✅ AI 활용 목표 12종 복수 선택 + 기타 자유 입력
5. ✅ 5단계 숙련도 (입문 → 전문가 자동화)
6. ✅ 자동 로드맵 생성 (Markdown, 한글 NanumGothic PDF)
7. ✅ Google Sheets 자동 저장 (SQLite 폴백)

### 알림·후속 처리
8. ✅ 강사에게 진단 제출 즉시 이메일 알림 (HTML, 시트 바로가기 버튼)
9. ✅ 수강생 본인 이메일로 진단 결과 PDF 자동 첨부 발송 (재진단 CTA 포함)

### 운영·관리자

    - KPI 4타일: 총 응답 / 최근 7일 / 평균 숙련도(1-5) / Top 매물

12. ✅ 전체 응답 테이블 + CSV 다운로드

    - 강의반: 입문(Lv1~2) / 기본 활용(Lv3) / 심화 자동화(Lv4~5)

12-3. ✅ **재진단 비교 카드** (2026-05-26 추가, 1차 — 결과 페이지) — 같은 이메일 재진단 시 변화 자동 시각화

    - 3개 컬럼: 숙련도(▲/▼ 단계차) · 주력 매물 변화(+추가/-제외) · AI 활용 목표 변화
    - "직전 진단으로부터 N일 경과" · "이 이메일로 N번째 진단" 메타 정보 표시
    - `find_previous_responses()`·`build_comparison()`·`render_comparison_card()` 순수 함수 분리
    - 매칭 키: **이메일** 만 사용 (상호·이름 달라도 매칭됨)
    - 알려진 제약: 사용자가 매번 다른 이메일을 입력하면 비교 불가 (다음 세션에서 안내 메시지 추가 검토)
    - ~~**🔧 UI 개선 (대표님 명시 요청, 2026-05-26)**~~ → **✅ 2026-05-27 완료 (A 단계)**:

12-4. ✅ **재진단 PDF 비교 페이지** (2026-05-27 추가, B-2 단계) — 첨부 PDF 첫 페이지를 디자인된 비교 리포트로

    - 숙련도 변화: 좌·우 표 (이전 vs 현재, 테두리·헤더) + 가운데 큰 ▲/▼ 단계차 강조
    - 추가/제외/유지 컬러 배지 (녹·빨·남) + 들여쓰기 아이템 리스트
    - 매물·목표 섹션 사이 회색 구분선

    - KPI 4타일: 재진단자(N명, 전체 비율) · 평균 재진단 횟수 · 평균 숙련도 변화 · 최고 성장 단계
    - 숙련도 변화 분포 막대 차트 (녹색) + Top 5 성장자 표 (순위·사용자·이메일·변화·횟수)

      - 평균 숙련도 변화 -0.3 단계 (더닝-크루거 신호 가능성 — 학습 후 자기 평가 보수화)

    - `MAX_TOKENS = 1500` (한국어 JSON 응답 여유) + `stop_reason == "max_tokens"` 시 logger.warning
    - silent fallback — API 키 미설정·자유 입력 빈값·import 실패·네트워크·인증·모델·파싱 에러 모두 None 리턴, 사용자 영향 0
    - `llm_insights.py` 모듈 신설, `is_llm_active()` / `generate_custom_insights()` / `_build_prompt()` / `_parse_response()` 4개 함수
    - **다음 세션 추가 예정 (2단계, 약 2h)**: PDF 메일에 LLM 인사이트 페이지 추가 + 강사 알림 메일에 `instructor_note` 한 줄 포함
