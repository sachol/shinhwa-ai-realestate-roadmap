# 신화AI부동산 — 공인중개사 맞춤형 AI 학습 로드맵 생성기

> 제작: 신화AI부동산 (노제승)
> 최종 사용자: 공인중개사

## 작업 목록

### 1. 설문 UI (Streamlit)
- [ ] 공인중개사 대상 진단 설문 폼 작성
  - 주력 매물 종류 (아파트 / 상가 / 토지 / 빌라·다세대 등)
  - AI 활용 목표 (블로그 자동화 / 매물 분석 / 고객 관리 / 광고 카피 등)
  - 현재 AI 숙련도 (상 / 중 / 하)
  - (확장 여지) 업무 경력, 지역, 일평균 매물 등록 수 등
- [ ] 제출 버튼 및 입력 검증
- [ ] 신화AI부동산 브랜딩 (타이틀, 로고 영역, 안내 문구)

### 2. 로드맵 생성 로직
- [ ] 설문 답변 → 추천 AI 도구 매핑 룰 설계
- [ ] 숙련도별 학습 순서(입문 → 심화) 분기 로직
- [ ] 결과 텍스트 템플릿 (공인중개사가 바로 읽기 좋은 톤)

### 3. PDF 다운로드
- [ ] fpdf2로 결과 PDF 생성
- [ ] 한글 폰트 임베딩 (fpdf2 기본은 한글 미지원 → NanumGothic 등 등록 필요)
- [ ] 신화AI부동산 헤더/푸터, 발행일자
- [ ] 다운로드 버튼 연결

### 4. 데이터 저장 (선택)
- [ ] SQLite로 설문 응답 누적 저장 → 추후 수강생 트렌드 분석 활용

### 5. 배포 (선택)
- [ ] Streamlit Community Cloud 또는 자체 호스팅 검토
